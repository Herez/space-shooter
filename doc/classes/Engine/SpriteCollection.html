Class Engine::SpriteCollection
In: engine/sprite_collection.rb
Parent: Object

SpriteCollection objects store a hash of sprite lists. A sprite list is simply an array of same-class Sprite objects.

In the game, there is one sprite list for each different sprite class, ie Alien, Bullet, Ship, etc.
